Netflix Backend API

A simplified simulation of Netflix's core backend functionality

üìã Overview

This backend simulates Netflix's core logic including user authentication, subscription management, movie catalog, device limits, and playback control.

Tech Stack: Django REST Framework, JWT Authentication, PostgreSQL

Deployment: Render

How to Use:
  • Register and log in to get access and refresh tokens
  • Use Authorization: Bearer <ACCESS_TOKEN> header for protected endpoints
  • When the access token expires, use the refresh endpoint to get a new one

üîê Authentication

POST Register 
curl -X POST https://netflix-backend-lncx.onrender.com/auth/register/ \
-H "Content-Type: application/json" \
-d '{"username":"test","password":"pass"}'
POST Login 
curl -X POST https://netflix-backend-lncx.onrender.com/auth/login/ \
-H "Content-Type: application/json" \
-d '{"username":"test","password":"pass"}'
POST Refresh Token 
curl -X POST https://netflix-backend-lncx.onrender.com/auth/token/refresh/ \
-H "Content-Type: application/json" \
-d '{"refresh":"<REFRESH_TOKEN>"}'
POST Logout 
curl -X POST https://netflix-backend-lncx.onrender.com/auth/logout/ \
-H "Authorization: Bearer <ACCESS_TOKEN>"
